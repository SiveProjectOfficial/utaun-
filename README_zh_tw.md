# 【什麼是 Utaun？】
這是一款僅通過完全無生物的加算合成技術來生成 UTAU 專用 CV 音源的 Windows 軟體。它會在您的「文件」資料夾中自動生成音源 ZIP 檔案。

### 【特點】
* 運行後在「文件」中自動生成音源 ZIP。
* 通過完全無生物的加算合成生成元音（a、i、u、e、o、n）。
* 通過與外部子音行（Ka 行至 Pa 行）組合生成 CV 音源。
* 支援平假名別名 + 英語羅馬字別名。
* 自動生成 `oto.ini` / `character.txt` / `readme.txt`。

### 【利用條款】
禁止加工、改造或再發布軟體。音源生成物的權利歸您所有。
