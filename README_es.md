# 【¿Qué es Utaun?】
Software de Windows que genera bancos de voz CV para UTAU mediante síntesis aditiva. Crea automáticamente un archivo ZIP en tu carpeta de Documentos.

### 【Características】
* Generación automática de ZIP en Documentos.
* Creación de vocales (a, i, u, e, o, n) por síntesis aditiva pura.
* Combinación con consonantes externas (filas Ka a Pa).
* Generación interna de las filas m / n / y / w.
* Soporta alias en Hiragana y Romaji.
* Genera automáticamente `oto.ini`, `character.txt` y `readme.txt`.
* Convierte imágenes PNG a JPG.

### 【Instrucciones】
1. Descarga `Utaun.ver-1.0.4.zip` desde **Releases**.
2. Extrae el archivo.
3. Ejecuta `Utaun.exe` y sigue los pasos.
4. El archivo final aparecerá en tu carpeta de Documentos.
