# 【Utaun Nedir?】
Yalnızca eklemeli sentez kullanarak UTAU için CV ses bankaları oluşturan bir Windows yazılımıdır. Belgeler klasöründe otomatik olarak bir ZIP oluşturur.

### 【Özellikler】
* Belgeler klasöründe otomatik ZIP oluşturma.
* Eklemeli sentez ile ünlü harf (a, i, u, e, o, n) üretimi.
* Harici ünsüzlerle (Ka-Pa arası) birleştirme.
* Hiragana ve Romaji alias desteği.
* `oto.ini`, `character.txt` ve `readme.txt` dosyalarını otomatik oluşturur.

### 【Şartlar】
Yazılımın değiştirilmesi, ticari kullanımı veya yeniden dağıtılması yasaktır. Üretilen sesler kullanıcıya aittir.
