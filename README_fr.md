# 【Qu'est-ce que Utaun ?】
Un logiciel Windows générant des banques de voix CV pour UTAU par synthèse additive. Un fichier ZIP est créé automatiquement dans votre dossier Documents.

### 【Caractéristiques】
* Génération automatique de ZIP dans Documents.
* Création de voyelles (a, i, u, e, o, n) par synthèse additive pure.
* Combinaison avec des consonnes externes (lignes Ka à Pa).
* Génération interne des lignes m / n / y / w.
* Support des alias Hiragana et Romaji.
* Création automatique de `oto.ini`, `character.txt` et `readme.txt`.

### 【Utilisation】
1. Télécharger `Utaun.ver-1.0.4.zip` via **Releases**.
2. Extraire le ZIP.
3. Lancer `Utaun.exe` et suivre les instructions.
4. Votre banque de voix est prête dans le dossier Documents.

### 【Conditions】
Modification, usage commercial et redistribution du logiciel interdits. Vous êtes libre de définir les règles pour vos banques générées.
