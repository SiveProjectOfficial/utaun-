# 【Co to jest Utaun?】
Oprogramowanie Windows do generowania banków głosowych CV dla UTAU przy użyciu syntezy addytywnej. Automatycznie tworzy plik ZIP w folderze Dokumenty.

### 【Funkcje】
* Automatyczne generowanie ZIP w Dokumentach.
* Tworzenie samogłosek (a, i, u, e, o, n) przez syntezę addytywną.
* Współpraca z zewnętrznymi spółgłoskami (rzędy Ka-Pa).
* Obsługa aliasów Hiragana i Romaji.
* Automatyczne tworzenie `oto.ini`, `character.txt` i `readme.txt`.

### 【Zasady】
Modyfikacja, użytek komercyjny i redystrybucja oprogramowania są zabronione. Prawa do wygenerowanych banków należą do Ciebie.
