# 【Ano ang Utaun?】
Windows software na gumagawa ng UTAU CV voice banks gamit ang additive synthesis. Awtomatikong gagawa ng ZIP file sa iyong Documents folder.

### 【Mga Tampok】
* Auto-generate ng ZIP sa Documents folder.
* Paggawa ng vowels (a, i, u, e, o, n) sa pamamagitan ng additive synthesis.
* Pwedeng pagsamahin sa external consonants (Ka-row hanggang Pa-row).
* Awtomatikong paggawa ng m / n / y / w rows sa loob ng app.
* Support para sa Hiragana at Romaji aliases.
* Awtomatikong paggawa ng `oto.ini`, `character.txt`, at `readme.txt`.

### 【Paano Gamitin】
1. I-download ang `Utaun.ver-1.0.4.zip` sa **Releases**.
2. I-extract ang ZIP.
3. Patakbuhin ang `Utaun.exe` at sundin ang instructions.
4. Ang ZIP ay mabubuo sa iyong Documents folder.

### 【Kasunduan】
Bawal baguhin, ibenta, o i-distribute ang Utaun.exe. Sayo ang karapatan sa mga mabubuong voice bank.
